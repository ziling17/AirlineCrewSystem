# Airline Crew Management System

This repository contains the **Airline Crew Management System** project, including source code, UML diagrams, and documentation. The project is implemented in Java and follows an object-oriented design.

---

## **Project Structure**

AirlineCrewSystem/
├─ src/ # Java source code
│ ├─ airline/ # Main package
│ │ ├─ database/ # Database class for storing model objects
│ │ ├─ management/ # Management classes (CrewManagement, FlightManagement, LeaveManagement, FlightHoursTracking)
│ │ ├─ model/ # Model classes (Person, Crew, Flight, Leave, CrewStatus)
│ │ └─ notification/ # Notification interfaces and implementations (Notifier, ConsoleNotifier, EmailNotifier)
├─ bin/ # Compiled .class files
├─ diagram/ # Diagrams illustrating system design
│ ├─ class/ # UML class diagrams showing relationships, inheritance, aggregation, and interfaces
│ ├─ sequence/ # Sequence diagrams showing the flow of operations
│ └─ usecase/ # Use case diagrams depicting system functionalities and actors
└─ README.md # Project overview, folder hierarchy, and usage instructions


---

## **Description of Each Folder**

src/ – Contains all Java source code organized by packages for database, management, model, and notification modules.
bin/ – Compiled Java .class files generated automatically by the IDE.
diagram/ – Contains diagrams illustrating system design:
class/ – Class diagrams showing the structure and relationships between classes
sequence/ – Sequence diagrams showing interactions between objects for key operations
usecase/ – Use case diagrams showing system functionalities and actors
README.md – Provides project overview, folder hierarchy, and instructions on using the project.
